Ranzz API adalah penyedia layanan REST API yang berfokus pada kecepatan, keandalan, dan kemudahan penggunaan. Dibangun dengan Express.js, melayani berbagai fungsi seperti Downloader, AI, dan Utilitas untuk developer.

TikTok Downloader

Mengunduh video dari TikTok tanpa watermark menggunakan URL video.

GET /api/downloader/tiktok?url={url_video}

Query Parameter

Parameter Tipe Wajib Deskripsi
url String Ya URL video TikTok yang valid.
